发表于 2017-03-29 | 分类于 工作和技能 | 0 comments

scp的基本使用有的时候是不够的，在这里陆续记录用到的一些关键字参数。

端口不是22的情况

用-P，大写。

总是出现stalled的情况

这是因为scp常常尽量使用最大带宽，那么网络环境不允许的情况下就会受限出现stalled情况。出现此提示的时候速度会降的很低。缓解这个问题的办法是人为限速。

这里的8192的单位是Kbit/s，也就是1兆每秒。这也是此设置能够达到的最大速度，只能往小了设置。
