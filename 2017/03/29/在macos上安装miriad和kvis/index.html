在macos上安装miriad和kvis

发表于 2017-03-29 | 分类于 工作和技能 | 0 comments

这俩软件在射电天文领域是很常用的。本文给出我安装的过程。

kivs

去kvis官网看看，在http://www.atnf.csiro.au/computing/software/karma/下载相关的karma压缩包。按要求解压，并按照ftp://ftp.atnf.csiro.au/pub/software/karma/README给出的后续步骤做就可以了，没碰着什么问题。
唯一的就是，kvis试运行如果出现了“localhost”连接不上什么的这种，那应该就是Xquartz版本不对了。我就是因为这个，先把它升级一下，再敲kvis就能正常启动了。

miriad

这个我之前尝试用官网的方式安装失败了，找了很久原因，依然不知道哪儿出的问题。
今天google的时候发现了这里https://sites.google.com/site/yoshitoshimajiri/memo-for-researches/miriad是个日文网站，但是里面出现了个port，灵机一动，如果能这么安装估计不会出现什么依赖之类的问题，说不定问题就解决了。

什么，你没有macport?

是的我惊奇的发现敲port的时候显示command not found.于是只能先装macport了。
去官网下载pkg安装包，双击运行安装即可。
