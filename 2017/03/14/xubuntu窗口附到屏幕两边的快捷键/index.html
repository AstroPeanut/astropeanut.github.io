xubuntu窗口附到屏幕两边的快捷键

发表于 2017-03-14 | 分类于 工作和技能 | 0 comments

整理onenote快速笔记的本子时发现的。当时google了很久但是都没解决，最终是自己冷不丁发现了的解决方案。所以记录在此备查。

我这个Dell机子因为只能装xfce的ubuntu，所以折腾了好一阵子才开始舒服起来。先是桌面窗口系统不好看，所以根据网上的步骤美化了一下，装了docky。然后感觉默认的终端程序太丑，所以装了gnome-terminal，并且折腾了好一阵子才懂得怎么把它设置成默认终端。
但是终端每次一启动总是很小一点，我的编程习惯总是全屏幕开终端窗口，左半边一个右半边一个。这就使得我每次启动了一个终端总要用鼠标拖动，很烦。因为ubuntu里面默认是ctrl+super+左右箭头就行了，而windows系统更是super+左右箭头就可以。
最终的解决方案是，在windows manager 里面，有四栏，style, keyboard, focus，advanced。在keyboard里面仔细看，发现是有相应的功能的，不过默认没有指定快捷键。所以指定一下就好了。仔细看里面有关于窗口的所有快捷键设置，包括最大化窗口等。

写这个笔记的时候Dell机子我还装不上ubuntu或者deepin啥的呢，如果当时知道是u盘boot的格式问题的话，啥都能装的。
