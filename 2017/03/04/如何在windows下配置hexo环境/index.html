如何在windows下配置hexo环境

发表于 2017-03-04 | 分类于 工作和技能 | 0 comments

平时写博客都是在苹果本上的Dropbox文件夹下进行，其动机是希望在另外的几个电脑上工作的时候也可以随时更新博客。但我随后发现了问题，似乎不同平台上的一些依赖是不一样的。这导致了我在不同电脑上更新文章会报错、不能正常运行hexo或者发生文件冲突。于是改变策略，在不同笔记本上新建一个blog文件夹，里面配置好了以后，每次更新的时候从dropbox相应文件夹下面拷贝文档进来，同时在本地的blog文件夹下面运行hexo generate && hexo server.这样一来，无论在哪个机器上，修改文档都在Dropbox下进行，但是博客生成的步骤只发生在本地blog文件夹下面。防止以后踩坑，把windows下面配置hexo环境的步骤记录一下。
