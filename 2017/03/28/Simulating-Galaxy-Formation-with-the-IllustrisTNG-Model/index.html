Simulating Galaxy Formation with the IllustrisTNG Model

发表于 2017-03-28 | 分类于 工作和技能 | 0 comments

这是Mar28的ICRAR/UWA Journal Club上讲的文章。这篇文章主要是介绍了Illustris数值模拟里面的model进一步升级的计划。

这个物理模型，基于moving mesh程序AREPO，考虑了大尺度上星系的形成和演化，考虑了引力+磁流体动力学的作用。这次改进是基于Illustris模拟的巨大成功基础上的，模型包含了恒星形成、恒星演化、化学演化、原初和金属线的cooling机制、恒星反馈、星系outflow、黑洞形成和生长以及其多模式的反馈机制等。这篇文章给出的是IllustrisTNG(The Next Generation)的物理核心。
这次对于星系的wind做了较大的改动。我们改动了其指向、速度、热学成分、能量计算，并对其与星系的族的影响做了进一步调查。
另外，之前一篇我们的系列文章提到，这次还引入了一个新的黑洞模型，以及磁流体动力学模型。
我们已经对这个新的模型进行了一些初期模拟进行试验，发现是可行的。并且，显示出了磁场对于恒星质量大于$10^{12}M_\odot$的暗晕有很大的影响。在一个就是，这次改正过的galaxy wind模型，解决了之前Illustris出现的低质量星系的尺度以及恒星组分方面的问题。
注意本文以及接下来将出现的一系列文章，还是在探索数值模拟用以改进的参数空间，用的盒子还是比较小的，主要精神都是在测试。

本文认为，磁场可能是比较重要的，而且galaxy wind可能是比较重要的，以及，AGN的反馈机制也可能有考虑的必要。
