写作计划

发表于 2017-02-17 | 分类于 生活和提高 | 0 comments

打算启动一个写作计划。

动机

  1. 首先是自己前段时间兴趣驱使下建了个个人博客网站，自己也通过使用体会到这是个记录经验和知识的好地方。
  2. 再就是，最近的POW+开始由Ahmed带领下开始看galaxy formation方面的书了。一本一本的来，计划中的进度还挺快的。我准备恶补知识的想法已经由来已久了，所以对我来说是个好消息。看这种书顺便用写作的方式帮助自己捋一捋必定时间好事。
  3. 本来也打算自己多练习练习写作。根据我之前的分析，写作的能力是很有用的，不管我将来是否依然在天文研究领域之内。
  4. 现在搬了家，通勤需要骑自行车，大约20分钟时间。考虑到骑车要考虑的天气状况，尤其是避免气温过高的时候太阳晒伤皮肤，已经计划了早晨很早出门晚上稍晚点回。这样一来可以把晚上专门安排出一个写作的时间，也算是个很好的总结和积累。

一些初步想法

  1. 平时看的文章，总结出来。比如之前在JC看的文章。
  2. 平时听的报告，要录音，并争取要到ppt，总结出来。
  3. 在astrobite，以及aasnova看到的文章，结合原始学术文献，总结出来。这个的好处是，既然已经有人先总结了，肯定会让我省些力气。
  4. 看的书做笔记。不管中文还是英文。当然POW+看的书的优先级肯定排在最前面。标准是POW+的书一定要搞定，其他的书籍不着急看，而要把它们当成参考文献查阅。
  5. JC是每周二开始，在周一晚上的时候争取多获取一些JC文章的信息。周二当天尽可能写一些JC文章的相关总结。
  6. 看NASA的每日一图这种材料，并且结合其图片说明给以总结，同时可以根据兴趣做相关的调查补充，甚至适当研读所对应的文献。这个的好处是保持一个比较好的物理/天文感觉，同时也增长了知识。如https://www.nasa.gov/image-feature/goddard/2016/hubble-uncovers-a-mysterious-hermit
  7. 中文也已经有了比较好的信息平台，最近出现的赛先生的天文专栏，由毛老师领头，看过几篇受益匪浅，可以同时关注一下并做笔记。
