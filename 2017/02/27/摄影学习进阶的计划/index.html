摄影学习进阶的计划

发表于 2017-02-27 | 分类于 兴趣爱好 | 0 comments

我的onenote里面设置了个TODO list，这个list里面有一条“学习和练习摄影”已经躺在那儿很久了，但是自知非常久没有去研究拍照或者至少看相关资料补充自己了。我自认为对摄影的兴趣并没有减少，但是生活上的杂事儿让我每天都感觉焦头烂额顾不上它。今天进行了一点反思，认为还是应该有一个纲领性的计划，像这种模棱两可的任务项放在TODO list里面意义是不可能太大的。

上网查了下别人关于摄影学习进阶的计划设定，发现了几个还算不错的idea,先放在这里并试行，后续如果有需要就不停的调整并在此文中做个更新记录，毕竟无论如何有个计划才好开始。

总体来说，我觉得以下这几个想法都很好，而且后几个想法实际上是可以合并到第一个里面的。启动一个每日拍摄一张照片的计划，在ins上面建个相册搞这个，并且用点儿心，把后几个想法也结合进去，想来会有些用处。

365每日拍摄计划

简单粗暴，每天拍一张，并发表到社交网络什么的。
这个方案有意思的地方有两个，一个是一次一张照片，意味着拍的时候以及后期选择的时候有所取舍，这对于培养自己摄影时的思路应该是有益的。另一个，是坚持每天都拍，可以起到记录生活的作用。这个不必非得是用相机拍，（当然我有相机肯定尽可能用相机，）不方便的话就用手机。这样也就杜绝了可能出现的大部分借口。

但是关于发表的社交网络…发ins可能还是比较合适的。朋友圈不太想搞这个。

每月一个主题

这个想法是说，每个月采用一个固定的主题来拍摄。这样的好处是可以让自己不听尝试不同的风格，拓宽自己可控的不同风格之外，对扩展自己的思路当然也大有好处。
这个想法当然也不错，不过我觉得这个完全可以和每日拍摄计划结合起来弄，而且也不用非得固定一个月一个主题，就只是有意识的一段时间内钻研一个感兴趣的主题或者风格就很好了。

特别摄影计划

一些比较特殊的拍摄风格的尝试。我觉得对我来说，典型的就是星空摄影。更偏一些的像延时摄影啥的暂时没啥兴趣。

模仿佳作

就是想办法“山寨”，多多模仿佳作。
