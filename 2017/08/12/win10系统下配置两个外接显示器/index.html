win10系统下配置两个外接显示器

发表于 2017-08-12 | 分类于 工作和技能 | 0 comments

今天终于找到了一个完美解决俩外接显示器正确显示的步骤，记录在此备查。

之前已经有了一个比较老的dell显示器，另外最近又淘了一个benq高清显示器。我把办公区域挪到沙发附近，两个显示器往那一放，把笔记本电脑合上不用自带显示器，外接键盘和无线鼠标，很舒服。但是每次在这个windows10下面配置使用另外两个显示器，而且要求分辨率正确总是很难。每次都要折腾好半天，然后冷不丁突然发现搞定了。当然一旦重启又一切照旧，不过平时工作完毕就睡眠也不错。烦人的是有时候因为更新等不得不重启，于是又很挠头。我估计这里让系统困扰，显得很“不智能”的原因，是我并不想让笔记本作为主显示器，甚至不想让它亮起，而另一方面我的显卡经过查证，只能同时显示两个显示器。这两个结合起来可能就够它受的了…没事儿，我理解…

  1. 在机器启动的时候，保证只有HDMI的benq一个外接，VGA的dell显示器的线拔掉。
  2. windows10进入系统后，不经过配置很可能笔记本+benq显示器的显示都达到了最优(如果没有的话，原则上简单配置可以搞定，如果分辨率不对的话一次重启应该可以解决)。
  3. 这时按下win+p，选择screen2 only。这个时候笔记本黑屏。
  4. 插上dell的VGA接线。这个时候可能自动跳成了extend，没关系，在win+p里面再选择screen2 only，这时候两个外接显示器均是最佳显示亮起，同时笔记本黑屏。合上笔记本盖子即可。

这里的第三步很可能是多余的。看起来在win+p这相关的底层设计里，我外接的这俩显示器被冠以screen2一个名字。

这里记录一下三个显示器的分辨率

  1. 笔记本 $1366\times$
  2. DELL显示器 $1280\times 1024$
  3. BENQ显示器 $1920\times 1080$
