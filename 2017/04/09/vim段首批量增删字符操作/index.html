vim段首批量增删字符操作

发表于 2017-04-09 | 分类于 工作和技能 | 0 comments

这里记录使用vim进行段首批量操作的技巧。

起因是写博客，正在读的一本书在网上可以找到目录描述。我于是把这些文字复制下来到atom编辑器，但是我希望能把它按照markdown的语法重写。琢磨了一会无果，想到了使用vim编辑器操作，发现可行。

以《费曼物理学讲义》这本书为例。在https://book.douban.com/subject/1437852/复制目录列表到vim新建的一个文件里面。

  • 去掉每行开头的章节号码

先gg到文件头，按ctrl+v进入visual模式，然后G到文件尾。左右挪动光标，选定后按d删除。

  • 调整

因为1-9章和后面的10-xx章占据字符是不一样的，所以后面一部分需要再调整一下。
先把光标放在需要再删除一个空格的起始行开头处，然后ctrl+v进入visual模式，然后G到文件尾，按d删除。

  • 每行加两个#

这里用到的语法是，用

为每行行首加入字符串

为每行行尾加入字符串。
解释一下，%为被编辑文件的每一行进行后续操作，$代表一行的结尾，^代表一行的开头。

  • 一级目录处删掉一个空格和一个#

这样发现一级目录多了一个空格，正好把它和二级目录分开来。
%s/## /# /g
大功告成。

# 计算机 # linux # 代码 # vim
