linux的rename用法--批量更改文件名

发表于 2017-04-24 | 分类于 工作和技能 | 0 comments

今天因为工作需要，研究了如何在一个文件夹下面批量重命名所有文件。原则上写个脚本也不会太长，但是毕竟已经有了很好用的轮子，不需要自己再造了。这就是rename工具。用perl写的，在shell下一般找得到。

参考http://www.cnblogs.com/amosli/p/3491649.html

rename的好处是非常支持正则语法。

参数说明：

  • -v会显示修改成功的文件名
  • -n不会执行任何操作，用来测试rename过程，可以用这个参数先测试一下效果，没问题再进行实际的批量操作
  • -f强制修改，会覆盖同名文件

举例说明：

  • rename -v 's/ /_/g' * 表示，把文件夹下面所有文件名的空格换成下划线
  • rename 's/\.txt//' *.txt 表示，去掉所有txt后缀名
  • rename 'y/A-Z/a-z/' * 表示，文件名大写全部变小写
  • rename *.JPG *.jpg 表示文件后缀JPG变成JPG
